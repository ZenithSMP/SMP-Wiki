# Skills

{% hint style="info" %}
This page was generated with the help of AI. Therefore, it may not be completely accurate.
{% endhint %}

Skills are leveled by performing related activities (mining, combat, farming, etc).\
Each skill grants **stats** and/or **effects** as you level up.

### Commands

* `/skills` - Opens the main skills menu
* `/skills stats` - Shows your current skill levels and bonuses.
* `/skills top` - Displays the top players for each skill.
